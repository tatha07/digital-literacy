# Digital Literacy Project - Complete Documentation

This repository contains a comprehensive digital literacy curriculum consisting of 5 tasks covering essential concepts for modern learners.

---

## 📋 Project Overview

This project demonstrates fundamental knowledge of digital literacy, professional online presence, coding platforms, digital communication etiquette, and cybersecurity awareness.

---

## 📁 Project Structure

### Task 1: Digital Literacy Presentation
**File:** `Task 1/presentation.md`

An infographic-based presentation created using Canva that explains digital literacy concepts for students. The infographic covers:
- Definition of digital literacy
- Essential digital tools (Google Docs, GitHub)
- Safe internet practices
- Basic email etiquette
- Clean, visually engaging design with icons and minimal text

**Key Takeaway:** Presents complex digital concepts in a clear, creative, and accessible way.

---

### Task 2: Digital Portfolio
**File:** `Task 2/portfolio.md`

Documentation of a professional digital portfolio built across three platforms:

**Platforms Used:**
- **GitHub:** For storing and sharing coding projects with a profile README
- **LinkedIn:** For professional networking and career development
- **Stack Overflow:** For problem-solving and analytical skill development

**Purpose:** Building a strong professional online presence to attract recruiters and academic opportunities.

**Future Goals:** Upload projects to GitHub, expand professional network on LinkedIn, and develop problem-solving skills on Stack Overflow over the next four years.

---

### Task 3: Coding Platforms & Collaboration Tools
**File:** `Task 3/platforms.md`

Exploration of practical coding and collaboration tools:

**Tools Explored:**
- **HackerRank:** Coding practice platform used to complete beginner-level challenges and improve problem-solving skills
- **Google Forms:** Created a Digital Literacy Awareness Quiz with 5 questions (multiple-choice and short-answer formats)
- **Google Sheets:** Linked to Google Forms for efficient data collection and analysis

**Applications:** Ideal for academic project management, technical interview preparation, and collaborative surveys.

---

### Task 4: Email Etiquette & Digital Communication
**File:** `Task 4/email-etiquette.md`

Comprehensive guide on professional digital communication with two main sections:

**Part A - Email Examples:**
- Extension request email to a professor (proper formatting and tone)
- Internship inquiry email (professional approach)

**Part B - Social Media & Communication Guidelines:**

✅ **Do's:**
- Use professional language and tone
- Think before posting
- Keep profiles updated
- Respect others' opinions
- Protect privacy settings

❌ **Don'ts:**
- Share personal/sensitive information
- Post offensive content
- Spread misinformation
- Engage in online arguments
- Use unprofessional language

**Key Lesson:** Effective digital communication builds positive impressions and prevents misunderstandings, especially when contacting professors or applying for opportunities.

---

### Task 5: Cybercrime Awareness & Prevention
**File:** `Task 5/cybercrime.md`

Comprehensive cybersecurity awareness module with case study and prevention strategies:

**Part A - Case Study:**
- **Topic:** Phishing attacks
- Explanation of how phishing works (fake emails, urgent messages, credential harvesting)
- Why students are vulnerable targets
- Consequences: financial loss, identity theft, unauthorized access

**Part B - Prevention Checklist:**

🔒 **General Safety Tips:**
- Verify sender before clicking links
- Never share passwords or OTPs
- Use strong, unique passwords
- Enable two-factor authentication (2FA)
- Avoid public Wi-Fi for sensitive activities
- Keep software updated

💳 **UPI & Financial Safety:**
- Never share UPI PIN
- Verify payment requests
- Check UPI IDs carefully
- Avoid unknown QR codes

🚨 **Cybercrime Reporting:**
- Report at: https://cybercrime.gov.in
- Helpline: 1930

**Key Takeaway:** Understanding common threats like phishing is essential for personal and financial security online.
